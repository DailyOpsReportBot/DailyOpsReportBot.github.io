Download Code Generator

Create a download code, generate its SHA-256 hash, then paste the hash entry into assets/js/download-codes.js.

Static site limitation: GitHub Pages cannot securely validate private codes because all client-side files are public. This tool is useful for simple customer gating, demos, and low-risk access control. Use a backend or private release delivery for stronger protection.

How to add the code

  1. Generate a code and hash here.
  2. Open assets/js/download-codes.js.
  3. Add the generated object to the window.DORB_DOWNLOAD_CODE_HASHES array.
  4. Commit and push the website to GitHub.
